bitbucket_pr_viewer
===================

Trivial Ruby script to list Pull Requests in Bitbucket.

Installation
------------
1. `git clone git@github.com:burennto/bitbucket_pr_viewer.git`

2. `gem install bundler`

3. `bundle install`

4. Edit _config.yml_

Usage
-----

1. `ruby bbprs.rb` and follow the prompts.
